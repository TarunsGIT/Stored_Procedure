# Stored_Procedure

1)c3p0-0.9.2.1.jar
2)mchange-commons-java-0.2.3.4.jar
3)mysql-connector-java-5.0.8-bin.jar

Excecute the sql script attached with project:
/Stored Procedure/src/employee_dept.sql
